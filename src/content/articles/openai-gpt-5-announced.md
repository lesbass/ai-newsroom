OpenAI announced GPT-5 on June 8, 2026, describing it as their most capable language model to date. The model shows notable improvements in reasoning tasks and code generation.

## Key improvements

According to OpenAI's announcement, GPT-5 achieves:

- **32% higher accuracy** on the MMLU benchmark compared to GPT-4 Turbo
- **Improved multi-step reasoning** on mathematical proofs and logical deductions
- **Enhanced code generation** with support for larger codebases and better debugging

## Performance benchmarks

The technical report details performance across several standard benchmarks:

| Benchmark | GPT-4 Turbo | GPT-5 |
|-----------|-------------|-------|
| MMLU | 86.4% | 89.7% |
| HumanEval | 67.0% | 78.2% |
| GSM8K | 92.0% | 95.1% |

## Availability

GPT-5 is available immediately through the OpenAI API for developers. Consumer access via ChatGPT is expected to roll out over the coming weeks.

## Analysis

The improvements in reasoning capabilities suggest OpenAI has made progress on chain-of-thought prompting and verification techniques. The code generation improvements could accelerate adoption in development workflows.

However, independent evaluation of these claims remains pending. We will update this article as third-party benchmarks become available.
